Steam Game Recommendation App

There are a large number of games available on Steam, and people may have a hard time finding the game that they are potentially interested in. This web application will recommend Steam games to the user based on their input Steam game and customized searching criteria. Recommendations will be generated using our model and the user will be able to review all information for recommended games.
